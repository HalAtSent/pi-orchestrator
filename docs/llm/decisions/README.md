# Decisions

This directory is for deliberate decision records when the repo needs them.

Decision records do not become product authority by being placed here. A
decision record is authoritative only when it declares `status: authoritative`,
cites its source authority, and stays within the scope described by that record.

Use decision records for durable choices that affect product behavior,
architecture boundaries, persisted contracts, or operational procedure.

Do not use this directory for scratch notes, stale plans, or uncited summaries.
